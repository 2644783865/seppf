常 见 问 题 解 答

问：浏览器不能正常显示能效平台页面

答：目前能效平台支持谷歌浏览器，其他有谷歌内核的浏览器能登录，但不能保证显示正常。

问：用户输入用户名和密码，但登录不了系统

答：新用户需要联系能效平台技术支持人员，需要运维部门人员设置域权限后，方能申请产品用户访问权限和登录系统。         

      老用户登录不上能效平台，可能是当前用户修改过机器登录密码或电脑密码过期，告知技术支持人员帮助修复。

问：如何才能不需要手动关闭开发任务和需求

答：迭代版本发布升级完成，必须项目经理在关闭版本时，系统会提示是否同步关闭开发任务和需求，如果此时同步关   闭，开发任务和需求会联动关闭，不需要任务责任人手动关闭。有的任务没有联动被关闭，是因为没有纳入版本。

问：如何上报线上问题

答：线上问题建议由产品经理在能效平台上报，需要判断问题类型。如果是环境问题，由运维部门直接解决；如果是需求问题，那么产品经理需要修正需求转报PIR需求；如果是软件问题，需要产品经理在产品需求页面转报PIR需求。
